# rate-repository-app

Kurssin osan 10 lopputuloksena syntyi GitHub repositoryjen arviointiin käytettävävä mobiiliappi. Hyödyntää React Nativea, Expoa ja GraphQL:ää. Appi löytyy skannaamalla Expo Go:lla oheinen QR-koodi. App on luotu versiolla SDK@55 eli jotta se avautuu, Expo Go:n tulee myös tukea kyseistä versiota.
